Impossible Maze

Team Members: Amos Sng, Zhi Ying, Edwin, Zaina and Yu Quan

Description:
Using Python (and its default libraries), our team have designed an educational game for students to build on different skills. When the user start the game, a maze will be randonly generated and players are required to clear different challenges to move forward within the maze. The challenges involve the following:

  • Number Sequence
  • Mathematics
  • Typing Test
  • Geography

Skills Involved: Python
